

需求描述：
进行系统注册时通常需要输入用户名和密码，
其中用户名要求由大写字母开头的8个字母字符组成、

密码由这四种元素组成（数字、大写字母、小写字母、特殊字符），且必须包含全部四种元素；密码长度大于等于8个，小于20个字符。
密码由8个（含8）以上，小于20大小写字母，数字和特殊字符（!@#$%^&*()_+`-={}|[]\:";'<>?,./）组成。

满足要求，则提示“注册成功”；
密码不正确提示“密码不符合要求”；
用户名不正确提示“用户名不符合要求”。


编写2个函数，要求实现用户名检查和密码检查，
并设计最少的测试数据进行判定覆盖测试。

分析要求：
用户名：
由大写字母开头的8个 字母 字符组成
不能以数字开头

密码：
1：由8个（含8）以上，但是小于20（含20）个字符组成。
2：数字、大写字母、小写字母和特殊字符；	[0-9A-Za-z~!@#$%^&*()_+`-={}|[]\:";'<>?,./]
3：必须包含全部四种元素；
